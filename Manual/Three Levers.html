Keep Talking and Nobody Explodes Mod Three Switches

On the Subject of Three Switches

So did I just hear three distinct light switch clicks?

  • Every time one of the other modules is solved zero or multiple switches need to be flipped.
  • To determine what switches to flip, use the rulessets below. What ruleset is used is switched every solved module, starting with the first ruleset.
  • If a module is solved, but the switches are not in the correct configuration, a strike will be dealt.
  • After a certain percentage of all modules is solved, the indicator light below will turn on.
  • When this light is on, switch all switches to one side, or to the other side when they are on one side already, to complete the module.

Switch Instruction Set I

  • If the number of solved modules is a multiplication of 7, flip switch one.
  • If the first letter of the last resolved module is a vowel, flip switch three.
  • If all switches are on the same side, flip switch two.
  • If the solved module was on the timer's side, flip switch three.
  • If you flipped the third switch last time, flip switch two.
  • If you made no mistakes, flip switch one.

Switch Instruction Set II

  • If you have made a mistake, flip switch two.
  • If all the switches are not on one side, flip switch three.
  • If the last letter of the last resolved module is a vowel, flip switch one.
  • If the solved module was not on the timer's side, flip switch one.
  • If the number of solved modules is a multiplication of 9, flip switch three.
  • If you flipped the first switch last time, flip switch two.

Switch Instruction Set III

  • If only the middle switch is up, flip switch one.
  • If you flipped the second switch last time, flip switch two.
  • If the solved module was not on the timer's side, flip switch three.
  • If you have made a two or more mistakes, flip switch two.
  • If the last letter of the last resolved module is a number, flip switch three.
  • If the number of solved modules is a multiplication of 6, flip switch one.
